# 介绍

----

`circles-ui` 是一款仿 `vant-ui` 移动端 Vue UI组件库

## 特性

- 基于 `Vue.Js` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供常用组件，方便业务开发

## 浏览器支持

- 现代浏览器和 IE9 及以上
